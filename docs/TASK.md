## 開発ガイドライン

### コード品質
- **ファイルサイズ**: 500行以内を目安
- **命名規則**: 責務ベースの具体的な命名
- **型安全性**: TypeScript strict mode
- **テスト**: 必要に応じてVitest使用

### アニメーション方針
- **パフォーマンス**: 60FPS維持
- **GPU加速**: transform のみ使用
- **レスポンシブ**: 全デバイス対応
- **アクセシビリティ**: モーション配慮

### デプロイ戦略
- **プレビュー**: プルリクエスト毎に自動
- **本番**: main ブランチマージで自動デプロイ
- **監視**: Lighthouse CI で継続監視
